A Flask app where you can create classes, where you create lessons by uploading a video which gets transcribed using whisper. Then the lesson page displays the transcript below the video.

The transcript is connected to the video in that it highlights the portion that the video is up to, and you can select a portion of the transcript and jump to that place in the video.

Furthermore, you can add comments to sections of the transcript, and reply to those comments
